sung.pydantic_models

Pydantic models generated by datamodel-codegen using the Spotify API OpenAPI spec.

python -m datamodel_code_generator –url https://raw.githubusercontent.com/sonallux/spotify-web-api/refs/heads/main/fixed-spotify-open-api.yml –input-file-type openapi –output pydantic_models.py

I used

https://raw.githubusercontent.com/sonallux/spotify-web-api/refs/heads/main/fixed-spotify-open-api.yml

because that version claims to be more correct than the official Spotify API spec:

https://developer.spotify.com/reference/web-api/open-api-schema.yaml
